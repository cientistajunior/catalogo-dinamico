# 📊 **Controle de Estoque:**
    Mecanismo dinâmico de busca, filtragem e renderização de dados na DOM.
## 🚀 Funcionalidades (Features)
* **Filtragem por Categoria:** Permite ao usuário filtrar a lista de produtos exibindo apenas itens de uma categoria específica (como "Celulares").
* **Busca Textual Dinâmica:** Filtra os produtos com base no texto digitado no campo de pesquisa.
* **Renderização Dinâmica:** Injeta e atualiza os elementos HTML na tela em tempo real utilizando JavaScript, sem necessidade de recarregar a página.
